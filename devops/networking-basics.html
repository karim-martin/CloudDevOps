Networking Basics

OSI Model

The OSI (Open Systems Interconnection) model has 7 layers:

LayerNameExamples
7ApplicationHTTP, HTTPS, FTP, SSH, DNS
6PresentationSSL/TLS, JPEG, GIF
5SessionNetBIOS, RPC
4TransportTCP, UDP
3NetworkIP, ICMP, ARP
2Data LinkEthernet, MAC
1PhysicalCables, Hubs

IP Addressing

IPv4

32-bit address, written as four octets (e.g., 192.168.1.1)

ClassRangePrivate Range
A1.0.0.0 - 126.255.255.25510.0.0.0/8
B128.0.0.0 - 191.255.255.255172.16.0.0/12
C192.0.0.0 - 223.255.255.255192.168.0.0/16

CIDR Notation

192.168.1.0/24   # 256 addresses (192.168.1.0 - 192.168.1.255)
10.0.0.0/16      # 65,536 addresses
172.16.0.0/12    # 1,048,576 addresses

DNS (Domain Name System)

TypePurposeExample
