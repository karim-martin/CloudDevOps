Helm Charts

Helm is the package manager for Kubernetes. It uses Charts to define, install, and upgrade applications.

Helm Commands

CommandDescription
helm repo add name urlAdd chart repository
helm repo updateUpdate repositories
helm search repo keywordSearch for charts
helm install name chartInstall chart
helm upgrade name chartUpgrade release
helm uninstall nameUninstall release
helm listList releases
helm status nameRelease status
helm history nameRelease history
helm rollback name revisionRollback release

Installing Charts

# Add repository
helm repo add bitnami https://charts.bitnami.com/bitnami
helm repo update

# Search for chart
helm search repo nginx

# Install chart
helm install my-nginx bitnami/nginx

# Install with custom values
helm install my-nginx bitnami/nginx -f values.yaml

# Install with inline values
helm install my-nginx bitnami/nginx --set replicaCount=3

# Install in namespace
helm install my-nginx bitnami/nginx -n production --create-namespace

# Dry run
helm install my-nginx bitnami/nginx --dry-run

Chart Structure

mychart/
├── Chart.yaml          # Chart metadata
├── values.yaml         # Default values
├── charts/             # Dependencies
├── templates/          # Kubernetes manifests
│   ├── deployment.yaml
│   ├── service.yaml
│   ├── ingress.yaml
│   ├── _helpers.tpl    # Template helpers
│   └── NOTES.txt       # Post-install notes
└── .helmignore

values.yaml

# values.yaml
replicaCount: 3

image:
  repository: nginx
  tag: "1.25"
  pullPolicy: IfNotPresent

service:
  type: ClusterIP
  port: 80

ingress:
  enabled: true
  host: myapp.example.com

resources:
  limits:
    cpu: 100m
    memory: 128Mi

Template Example

# templates/deployment.yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: {{ include "mychart.fullname" . }}
spec:
  replicas: {{ .Values.replicaCount }}
  selector:
    matchLabels:
      app: {{ include "mychart.name" . }}
  template:
    spec:
      containers:
      - name: {{ .Chart.Name }}
        image: "{{ .Values.image.repository }}:{{ .Values.image.tag }}"
        ports:
        - containerPort: {{ .Values.service.port }}

Creating Charts

# Create new chart
helm create mychart

# Lint chart
helm lint mychart

# Template locally
helm template mychart

# Package chart
helm package mychart

# Install local chart
helm install myrelease ./mychart
