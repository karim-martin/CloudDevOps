Service & Package Management

12 min read

Service Management (systemd)

Most modern Linux distributions use systemd to manage services. The systemctl command is used to control services.

Command Description
systemctl start service Start a service
systemctl stop service Stop a service
systemctl restart service Restart a service
systemctl reload service Reload configuration
systemctl status service Check service status
systemctl enable service Enable service on boot
systemctl disable service Disable service on boot
systemctl is-active service Check if service is running
systemctl is-enabled service Check if service is enabled
systemctl list-units --type=service List all services 
# Example: Managing nginx
sudo systemctl start nginx
sudo systemctl enable nginx
sudo systemctl status nginx

Viewing Logs (journalctl)

Command Description
journalctl View all logs
journalctl -u service View logs for specific service
journalctl -f Follow logs in real-time
journalctl -u service -f Follow service logs
journalctl --since "1 hour ago" Logs from last hour
journalctl -p err Show only errors
journalctl -b Logs since last boot

Package Management: APT (Debian/Ubuntu)

Command Description
apt update Update package index
apt upgrade Upgrade all packages
apt install package Install package
apt remove package Remove package
apt purge package Remove package and config
apt autoremove Remove unused dependencies
apt search keyword Search for packages
apt show package Show package details
apt list --installed List installed packages
dpkg -i package.deb Install local .deb file 
# Example: Installing nginx on Ubuntu
sudo apt update
sudo apt install nginx -y

Package Management: YUM/DNF (RHEL/CentOS/Fedora)

YUM vs DNF

DNF is the next-generation version of YUM. Fedora and RHEL 8+ use DNF by default. Commands are mostly interchangeable.

Command Description
yum check-update Check for updates
yum update Update all packages
yum install package Install package
yum remove package Remove package
yum search keyword Search for packages
yum info package Show package info
yum list installed List installed packages
yum clean all Clean cache
rpm -ivh package.rpm Install local .rpm file 
# Example: Installing nginx on RHEL/CentOS
sudo yum install epel-release -y
sudo yum install nginx -y

Process Management

Command Description
ps aux List all processes
ps aux | grep name Find process by name
top Interactive process viewer
htop Enhanced process viewer
kill PID Terminate process
kill -9 PID Force kill process
killall name Kill all processes by name
pgrep name Find PID by name
pkill name Kill by name
nohup command & Run in background
