Network Management

10 min read

Network Information

Command Description
ip addr Show IP addresses and interfaces
ip link Show network interfaces
ip route Show routing table
ifconfig Show interfaces (legacy)
hostname Show system hostname
hostname -I Show all IP addresses

Testing Connectivity

Command Description
ping host Test connectivity to host
ping -c 4 host Send 4 ping packets
traceroute host Show route to host
mtr host Combined ping and traceroute
curl -I url Fetch HTTP headers
wget url Download file from URL

DNS Commands

Command Description
nslookup domain Query DNS for domain
dig domain DNS lookup with detailed info
dig +short domain Short DNS lookup
host domain Simple DNS lookup
cat /etc/resolv.conf View DNS configuration
cat /etc/hosts View local hosts file

Port and Connection Info

Command Description
netstat -tuln Show listening ports
ss -tuln Show listening ports (modern)
ss -s Socket statistics summary
lsof -i :80 Show process using port 80
nc -zv host port Test if port is open
telnet host port Connect to port (testing)

Firewall (iptables/firewalld)

iptables

Command Description
iptables -L List all rules
iptables -A INPUT -p tcp --dport 80 -j ACCEPT Allow port 80
iptables -A INPUT -s IP -j DROP Block IP address
iptables -F Flush all rules

firewalld (RHEL/CentOS)

Command Description
firewall-cmd --list-all Show current configuration
firewall-cmd --add-port=80/tcp --permanent Open port 80
firewall-cmd --reload Reload firewall rules

ufw (Ubuntu)

Command Description
ufw status Show firewall status
ufw enable Enable firewall
ufw allow 80/tcp Allow port 80
ufw deny 22 Deny port 22

SSH Commands

Command Description
ssh user@host Connect to remote host
ssh -p 2222 user@host Connect on specific port
ssh -i key.pem user@host Connect with private key
ssh-keygen Generate SSH key pair
ssh-copy-id user@host Copy public key to remote host
scp file user@host:/path Copy file to remote host
scp user@host:/path/file . Copy file from remote host
SSH Config

Create ~/.ssh/config to save connection settings: 

Host myserver
    HostName 192.168.1.100
    User admin
    Port 22
    IdentityFile ~/.ssh/mykey

Then simply run: ssh myserver
