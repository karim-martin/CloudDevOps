Vim Editor

10 min read

Vim is a powerful text editor that comes pre-installed on most Linux systems. Learning Vim is essential for DevOps work, especially when editing configuration files on servers.

Vim Modes

Vim operates in different modes:

Mode Purpose How to Enter
Normal Navigation and commands Press Esc
Insert Typing text Press i, a, o
Visual Selecting text Press v
Command Saving, quitting, etc. Press :

Basic Commands

Opening and Closing

Command Description
vim filename Open file in Vim
:w Save file
:q Quit Vim
:wq Save and quit
:q! Quit without saving
:x Save and quit (same as :wq)
ZZ Save and quit (normal mode)

Entering Insert Mode

Key Description
i Insert before cursor
I Insert at beginning of line
a Append after cursor
A Append at end of line
o Open new line below
O Open new line above

Navigation

Key Description
h, j, k, l Left, down, up, right
w Jump to next word
b Jump to previous word
0 Go to beginning of line
$ Go to end of line
gg Go to first line
G Go to last line
:n Go to line n
Ctrl+f Page down
Ctrl+b Page up

Editing Commands

Key Description
x Delete character under cursor
dd Delete (cut) current line
dw Delete word
d$ Delete to end of line
yy Yank (copy) current line
yw Yank word
p Paste after cursor
P Paste before cursor
u Undo
Ctrl+r Redo
. Repeat last command

Search and Replace

Command Description
/pattern Search forward
?pattern Search backward
n Next match
N Previous match
:%s/old/new/g Replace all occurrences in file
:%s/old/new/gc Replace all with confirmation
:s/old/new/g Replace all in current line
Pro Tip

Run vimtutor in your terminal for an interactive Vim tutorial that takes about 30 minutes to complete.

Visual Mode

Key Description
v Start visual mode (character)
V Start visual line mode
Ctrl+v Start visual block mode
d Delete selection
y Yank selection
> Indent selection
< Unindent selection

Useful Settings

" Add these to ~/.vimrc
set number          " Show line numbers
set relativenumber  " Show relative line numbers
set tabstop=2       " Tab width
set shiftwidth=2    " Indent width
set expandtab       " Use spaces instead of tabs
set autoindent      " Auto indent new lines
set hlsearch        " Highlight search results
set incsearch       " Incremental search
syntax on           " Enable syntax highlighting
