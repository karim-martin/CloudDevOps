GCP Cloud Services

Google Cloud Platform services for containerized workloads.

This section covers GCP services commonly used with containerized applications running on GKE and Cloud Run.

Compute

Container orchestration and serverless compute platforms.

GKE

Google Kubernetes Engine for container orchestration.

Cloud Run

Serverless containers with automatic scaling.

Databases

Managed database services for containerized applications.

Cloud SQL

Managed MySQL, PostgreSQL, and SQL Server.

Firestore

NoSQL document database for real-time data.

Memorystore

Managed Redis and Memcached for caching.

Storage

Object storage and container registries.

Cloud Storage

Object storage for any type of data.

Artifact Registry

Store and manage container images and packages.

Networking

VPC, load balancing, CDN, and DNS for containers.

VPC

Virtual Private Cloud for network isolation.

Load Balancing

Global and regional load balancers for GKE.

Cloud CDN

Content delivery network for faster performance.

Cloud DNS

Managed DNS service for domains.

Security

Identity, secrets, and application protection.

IAM

Identity and Access Management, Workload Identity.

Secret Manager

Secure storage for sensitive data.

Cloud Armor

DDoS protection and WAF.

DevOps

CI/CD services for containerized deployments.

Cloud Build

Serverless CI/CD for building containers.

Cloud Deploy

Managed continuous delivery to GKE and Cloud Run.

Observability

Monitoring, logging, and alerting for containers.

Cloud Monitoring

Metrics, dashboards, and alerting.

Cloud Logging

Centralized log management and analysis.
Container Focus

This section focuses on GCP services commonly used with containerized workloads on GKE and Cloud Run. For end-to-end deployment guides, see the App Deploy section.
