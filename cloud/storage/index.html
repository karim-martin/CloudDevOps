Storage Services

GCP storage services for container workloads - object storage for files and Artifact Registry for container images.

Storage Options

Cloud Storage

Object storage for files, backups, static assets, and data lakes.

Artifact Registry

Store and manage container images, packages, and build artifacts.

FeatureCloud StorageArtifact Registry

ContentAny files/objectsDocker images, npm, Maven, etc.
